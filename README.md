# jsproject2
HTML + CSS + JavaScript Mortgage Calculator Webapp

This web app calculates the monthly mortgage payment for a fixed-rate mortgage.
The user can enter the principal, the yearly rate as a percentage, and select the length of the mortgage. 
After that, click the calculate button to see the monthly mortgage payment for a fixed-rate mortgage.
